# Readme
基于.net core 2.2
在Webhost中实现定时任务，异步队列任务。基于 IHostedService ，在 通用主机中实现
通用主机是 ASP.NET Core 2.1 中的新增功能，不适用于 Web 承载方案。 对于 Web 承载方案，请使用 Web 主机。 通用主机将在未来版本中替换 Web 主机，并在 HTTP 和非 HTTP 方案中充当主要的主机 API
